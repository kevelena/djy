<h6>巴哈马美丽的海滨

<hr>
<p>【大纪元2019年07月24日讯】（记者张可欣温哥华工商报道）<span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">二战时，德国</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">Ennepetal</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">居民在克鲁特盐洞躲避空袭。神奇的是，呆在洞里的人，很少咳嗽，呼吸畅顺。后来科学家证实，当时进入盐洞的哮喘病、慢性支气管炎或其他呼吸道疾病的人，症状都得到了治愈，健康者也改善了免疫力。</span></span></span></p>
<p><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';"><a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E7%9B%90%E7%96%97.md">盐疗</a>法历史悠长。埃及艳后克丽奥佩脱拉（</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">Cleopatra</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">）女皇，以死海中的矿物盐和矿物泥来护肤养颜；近代中东战争中，以色列政府用死海里的矿物盐、矿物泥温泉疗法，来治疗战场上因受伤而皮肤发炎、溃烂等的患者。</span></span></span></p>

<h2><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">臭氧蒸汽疗法的机理</span></span></span></h2>
<figure id="attachment_11404753" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2019/07/Ozone-Steam-Sauna-SaltParadise-3-e1563905961448.jpg"><img class="size-full wp-image-11404753" src="http://i.epochtimes.com/assets/uploads/2019/07/Ozone-Steam-Sauna-SaltParadise-3-e1563905961448.jpg" alt="图：盐天堂的臭氧桑拿治疗室，为纯天然、非药物治疗法。（盐天堂提供）" width="600" b="400" /></a><figcaption class="wp-caption-text">图：盐天堂的<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E8%87%AD%E6%B0%A7%E6%A1%91%E6%8B%BF.md">臭氧桑拿</a>治疗室，为纯天然、<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E9%9D%9E%E8%8D%AF%E7%89%A9%E6%B2%BB%E7%96%97.md">非药物治疗</a>法。（盐天堂提供）</figcaption></figure>
<p><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">古代的罗马浴、瑞典桑拿浴、秘鲁的桑拿屋等，是人类应用高温法治愈疾病的鼻祖，后来因为其在癌症治疗上的特效，更受到广泛欢迎。</span></span></span></p>
<p><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">人类用臭氧治病，也有近</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">200</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">年的历史。</span></span></span></p>
<p><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">一战期间，德国医生利用臭氧，治疗外伤和厌氧菌感染的伤员。</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">1932</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">年德国牙医</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">Fishch</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">把臭氧作为抗感染药物。</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">1958</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">年德国人</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">JoaquimHansler</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">发明可控浓度的臭氧发生器。</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">1966</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">年，科学家</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">Otto warburg</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">教授关于肿瘤细胞具有厌氧性的研究，因此获得当年的诺贝尔奖。本世纪初，臭氧治疗在中国大陆也被广泛应用。</span></span></span></p>
<p><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">美国一位癌细胞严重扩散的前列腺癌病人，</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">2</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">个月内做</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">21</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">次臭氧蒸气浴，</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">PSA</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">指数由</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">1046</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">降到</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">64</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">。</span></span></span></p>
<p><span style="font-family: Arial;"><span lang="zh-CN"><span style="color: #252525;"><span style="font-family: 'Arial Unicode MS';">人坐在臭氧蒸汽桑拿舱内，手与头部外露，舱内的桑拿蒸汽环绕整个身体，臭氧也随着蒸汽进入皮肤与血液，再送到身体各处，触及到人体的脂肪与毒素积存的地方。</span></span><span style="font-family: 'Arial Unicode MS';">促进血液循环，帮助受伤的肌肉快速修复；刺激周围血管的血管扩张缓解疼痛并加速愈合过程；消除各种细菌和病毒感染；加速内部器官和内分泌腺的代谢过程。</span></span></span></p>
<p><span style="font-family: Arial;"><span lang="zh-CN"><span style="color: #252525;"><span style="font-family: 'Arial Unicode MS';">臭氧进入体内后，氧化排泄掉人体的有害毒素，比如杀虫剂、酸性物质等，让细胞呼吸到新鲜空气。病人体内排出的毒素，比如重金属等，有的从皮肤表面渗出，人体免疫能力也得到增强。</span></span></span></span></p>
<h2><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">重金属排毒</span></span></span><span style="font-family: 'Arial Unicode MS', serif;">(</span><span style="color: #252525;"><span style="font-size: medium;">ionic detoxification</span></span><span style="font-family: 'Arial Unicode MS', serif;">)</span><span style="font-family: Arial;"><span lang="zh-CN"><span style="font-family: 'Arial Unicode MS';">的机理</span></span></span></h2>
