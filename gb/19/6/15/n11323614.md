<p>1. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11323306.md">亲共派转向 无人撑林郑 传韩正等聚深圳</a><br />
香港百万人抗议港府提出的修订《<a href="https://github.com/dfh1/djy/blob/master/gb/tag/%E9%80%83%E7%8A%AF%E6%9D%A1%E4%BE%8B.md">逃犯条例</a>》，但特首林郑月娥仍坚持“不撤回、不暂缓”修法，引爆大规模警民冲突。日前港媒报导，有消息称中共政治局常委韩正南下深圳，会晤港澳官员，修例或无限期暂缓，但不撤。</p>
<p>2. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11323123.md">组图：六千港妈集会 抗议血腥镇压促撤恶法</a><br />
6?12港警镇压数万抗议民众事件后第三天，一群香港妈妈们不满林郑月娥“母亲论”， 14日晚上在遮打花园集会，抗议香港政府血腥镇压，要求撤销《引渡条例》。大会晚上九点宣布，共有六千人集会。</p>
<p>3. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/10/n11312610.md">遭受中共蹂躏的中国文艺界精英们（2）</a><br />
据明慧网统计，大陆文艺界法轮功学员自1999年7月遭中共迫害20年来，至少有9人被迫害致死，至少75人遭绑架，其中遭非法判刑或非法劳教迫害50人次。</p>
<p>4. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11321226.md">各界谴责港府将和平抗议定为“暴动”</a><br />
数以万计香港市民6月12日重返金钟，反对修订《<a href="https://github.com/dfh1/djy/blob/master/gb/tag/%E9%80%83%E7%8A%AF%E6%9D%A1%E4%BE%8B.md">逃犯条例</a>》，其后演变为冲突。警方将其定性为“暴动”，更发射橡胶弹、催泪弹等镇压示威者，引起各界强烈谴责。13日香港法律界、宗教界等团体发表声明或召开记者会，抗议警方过度使用武力。</p>
<p>5. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/13/n11318694.md">江泽民孙子江志成的捞钱路线图</a><br />

<p>6.<a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11322697.md">【新闻看点】港人抗争震撼全球 陆民翻墙围观</a><br />

<p>7. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11323362.md">专家：中共强推引渡条例 美国有能力反击</a><br />

<p>8. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11323283.md">英文大纪元报导美投资峰会 白宫推荐阅读</a><br />

<p>9. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11323220.md">外媒：忧引渡条例 香港富豪开始转移财富</a><br />

<p>10. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11323025.md">组图2：港警612镇压 雨伞抵抗催泪弹</a><br />

<p>11. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11322698.md">【新闻看点】六四将重现香港？习近平成关键</a><br />

<p>13.<a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11322606.md">大陆汽车经销巨头“庞大集团”宣布破产</a><br />

<p>14. <a href="https://github.com/dfh1/djy/blob/master/gb/19/6/14/n11322824.md">程晓容：香港怒吼 国际声援 中共恐慌耍赖</a><br />
