<a href="https://github.com/dfcbzs/djy/blob/master/gb/tag/%E5%B7%9D%E6%99%AE.md">川普</a>这次不仅出面制止中共对香港进行暴力维稳，同时还主动提出了和平化解香港问题的谈判建议，这一点不仅超出中共的盘算之外，恐怕也超出了许多国际关系学者的意料之外。可以说，川普又做了一次“不按牌理出牌”的典型示范。</p>
<p>2. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/16/n11458494.md">付国豪被指没记者证 军情身份或呼之欲出</a><br />
中共喉舌《环球时报》记者<a href="https://github.com/dfcbzs/djy/blob/master/gb/tag/%E4%BB%98%E5%9B%BD%E8%B1%AA.md">付国豪</a>遭香港示威者围攻后，他的身份一直是个谜。《环球时报》主编承认付国豪是该报记者，但说他没有记者证。大陆媒体人认为，其军情人员身份呼之欲出。</p>
<p>3. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/16/n11458852.md">中共卧底疑云 分析：图栽赃示威者引镇压</a><br />

<p>4. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/15/n11455881.md">追随中共迫害法轮功的校长们灾祸连连</a><br />

<p>5. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/16/n11458002.md">6万人遮打花园集会 促国际关注香港</a><br />

<p>6. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/15/n11454083.md">【新闻看点】警告香港勿重演六四 美有何王牌？</a><br />

<p>7. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/16/n11458480.md">香港年轻抗议者：必须直接向中共发出信息</a><br />

<p>9. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/12/n11448967.md">【新闻看点】卧底阴霾下 港人现场捉公安细节</a><br />

<p>10. <a href="https://github.com/dfcbzs/djy/blob/master/gb/19/8/16/n11458590.md">中共禁陆片参加金马奖 《少年问道》拒不退赛</a><br />
