<tr><td VALIGN=TOP width="626"><h2 align=center>湾区太阳能发电板 真正节省电费（一）</h2>
<img src="http://i.epochtimes.com/assets/uploads/2019/08/Kristin-Ridge-Wy-600x400.jpg" />
<h6>湾区太阳能公司Solar System Inc.保证太阳能板25年的质量，25年后还有80%的产电能力。（圣荷西太阳能Solar System Inc.提供）
</h6>
<hr>
<p>【大纪元2019年08月06日讯】文：李沁一?大纪元<br />
PG&amp;E不久前提出2020年的电费要涨10亿元，2021年再涨4亿5,400万元，2022年还要涨4亿8,600万元。面对不断上涨的电费，人们想到了很多节省的办法，其中之一就是在屋顶安装太阳能发电板，利用取之不尽、用之不竭的太阳能发电，<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E8%8A%82%E7%9C%81%E7%94%B5%E8%B4%B9.md">节省电费</a>。</p>
<p>笔者近期采访了<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E5%8A%A0%E5%B7%9E%E5%A4%AA%E9%98%B3%E8%83%BD.md">加州太阳能</a>公司Solar System Inc.，询问了该公司安装太阳能板的价格、质量，能节省多少电费，以及售后、维修、是否损害屋顶等问题。</p>
<p><a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E5%8A%A0%E5%B7%9E%E5%A4%AA%E9%98%B3%E8%83%BD.md">加州太阳能</a>公司Solar System Inc.的总经理Jason Lee（李记正）首先表示，公司的所有讲解、咨询服务免费；公司到用户家查看房屋状况免费；是否适合安装太阳能发电板等上门服务都免费。一直到用户同意购买产品，与公司建立合同关系后，才开始收费！</p>
<figure id="attachment_11432928" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2019/08/Kristin-Ridge-Wy.jpg"><img class="size-large wp-image-11432928" src="http://i.epochtimes.com/assets/uploads/2019/08/Kristin-Ridge-Wy-600x510.jpg" alt="" width="600" b="510" /></a><figcaption class="wp-caption-text"><a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E6%B9%BE%E5%8C%BA%E5%A4%AA%E9%98%B3%E8%83%BD.md">湾区太阳能</a>公司Solar System Inc.保证太阳能板25年的质量，25年后还有80%的产电能力。（<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E5%9C%A3%E8%8D%B7%E8%A5%BF%E5%A4%AA%E9%98%B3%E8%83%BD.md">圣荷西太阳能</a>Solar System Inc.提供）</figcaption></figure>
<p>接着Jason介绍说，太阳能发电技术安全稳定，已趋成熟。近年，加州政府都在推广安装太阳能板，对安装的用户实行现金奖励及税额抵免（参照www.csi-epbb.com）。</p>
<h2>买太阳能板 加州政府送返还金</h2>
<p>例如，某用户在2019年花费2万元购买太阳能板，加州政府返还两万元的30%（即6,000元），如果当年该用户交税超过6,000元，可以一次性拿到返还金6,000元；如果当年交税不足6,000元，加州政府将在5年内返还完毕。而且购买太阳能发电板的费用也能抵税。</p>
<p>但是要注意：返还费用2019年是30%，明年（2020年）减少为26%，2021年为22%，到2022年就没有了！所以计划安装的用户请把握好时机！</p>
<p>下面是关于在旧金山湾区安装Solar System Inc.太阳能公司的太阳能发电板，普遍存在的一些问答：</p>
<h2>一、安装Solar System Inc.公司的太阳能板能节省多少电费？</h2>
<p>答：以南湾A家庭举例，A家庭每年用电约8,000W，交电费约2,000元/年。如果安装太阳能板，一次性投资16,000～20,000元。</p>
<p>北加州的Solar System Inc.太阳能公司担保产品质量25年，保证25年后太阳能板还有80%的产电能力。25年内A家庭比传统用电节省122,567元，这是高达85%的省钱比例。</p>
<figure id="attachment_11432929" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2019/08/Jennifer-Khuu.jpg"><img class="size-large wp-image-11432929" src="http://i.epochtimes.com/assets/uploads/2019/08/Jennifer-Khuu-600x343.jpg" alt="" width="600" b="343" /></a><figcaption class="wp-caption-text">位于湾区圣荷西（San Jose）的太阳能公司Solar System Inc.，能在不同的屋顶架构太阳能板。（<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E6%B9%BE%E5%8C%BA%E5%A4%AA%E9%98%B3%E8%83%BD.md">湾区太阳能</a>Solar System Inc.提供）</figcaption></figure>
<h2>二、一般要装多少块太阳能板？</h2>
<p>答：太阳能板的板长约1.67米，宽1米，每块板一般发电325-330W/小时，好一些的可以达到发电370W/小时（比如加州Solar System Inc.太阳能公司经营的韩国LG品牌太阳能板）。还以南湾的A家庭为例，每年用电约8,000W，需要安装16～18块。</p>
<h2>三、哪种太阳能板售价高？</h2>
<p>答：加州南湾Solar System Inc.太阳能公司正在销售的太阳能板，产地来自新加坡、中国、美国、日本等地区，售价大约16,000元（发电能力约325-330W/小时），韩国LG品牌的售价约20,000元（发电能力约370W/小时）。</p>
<h2>四、用户都要有存电设备吗？</h2>
<p>答：用户不需要有存电设备。用户把白天产生的电能输送给PG&amp;E公司，由PG&amp;E储存。用户用电还和原来一样，使用从PG&amp;E输送过来的电。比如，用户白天给PG&amp;E输送10元的电，夜晚使用12元的电，那么该用户当日的电费就是2元。以此类推。</p>
<p>用户在使用<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E5%8C%97%E5%8A%A0%E5%B7%9E%E5%A4%AA%E9%98%B3%E8%83%BD.md">北加州太阳能</a>Solar System Inc.的产品之后的反馈：</p>
<p>南湾<a href="https://github.com/cbzs/djy/blob/master/gb/tag/%E5%9C%A3%E8%8D%B7%E8%A5%BF%E5%A4%AA%E9%98%B3%E8%83%BD.md">圣荷西太阳能</a>公司Solar System Inc.提供的太阳能板产品多样化，我们在认真听取介绍后决定购买。――D女士。</p>
<p>先后向十几家公司发出咨询函，别的公司要等2至3周才能约到，旧金山湾区的Solar System Inc.太阳能公司回复迅速，比较之后选择了他们。他们服务透明，上午开工，下午完成，我们非常幸运，当天就通过了验收！――Y先生。</p>
<p>Solar System Inc.成立5年，已有四百多家用户，提供华人服务，售后很好，一直跟进我们的使用情况。――C女士。</p>
