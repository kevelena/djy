<h1>蔡F佑河滨献唱 与众星齐聚“风格路跑 ”</h1>
<img src="http://i.epochtimes.com/assets/uploads/2019/04/1904201236391487-600x400.jpg" />

```diff
- 蔡F佑于4月20日出席“ELLE Run with Style风格路跑”活动。（ELLE提供）
```
<hr>
<p>【大纪元2019年04月21日讯】歌手<a href="https://github.com/szzd1/djy/blob/master/gb/tag/%E8%94%A1%E6%97%BB%E4%BD%91.md">蔡F佑</a>于昨（20）日受邀出席大佳河滨公园举办的第五届“ELLE Run with Style风格<a href="https://github.com/szzd1/djy/blob/master/gb/tag/%E8%B7%AF%E8%B7%91.md">路跑</a>”活动，今年吸引八千多名跑友及艺人们一同参与。今年路跑以“复古X音乐”为主题，由蔡F佑、旺福、严正岚等人接力开唱，蔡F佑一连带来四首歌曲。</p>
<figure id="attachment_11201158" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2019/04/1904201245071487.jpg"><img class="size-large wp-image-11201158" title="" src="http://i.epochtimes.com/assets/uploads/2019/04/1904201245071487-600x400.jpg" alt="" width="600" b="400" /></a><figcaption class="wp-caption-text"><a href="https://github.com/szzd1/djy/blob/master/gb/tag/%E8%94%A1%E6%97%BB%E4%BD%91.md">蔡F佑</a>（中）出席“ELLE Run with Style风格<a href="https://github.com/szzd1/djy/blob/master/gb/tag/%E8%B7%AF%E8%B7%91.md">路跑</a>”活动，并在大佳河滨公园献唱。（ELLE提供）</figcaption></figure>
<p>活动现场更有赛道三大音乐主题舞台DJ热力放送，打造最大型的晨间音乐运动派对，现场气氛热烈。而蔡F佑在演唱《好不好》时，还示范慢跑后拉筋伸展的动作，笑说要当全场腿最开的艺人。</p>
<p>唱到压轴歌曲《我可以》时，底下的观众也一起大合唱，让他感触很深，最后也送了一段感谢的话给现场朋友，“现在的我仍然朝着梦想前进，无论遇到阻碍或是想放弃时都一定会继续坚持下去，也希望这鼓力量可以分享给全场的朋友们。”</p>
<p>蔡F佑此次路跑活动挑战休闲组4.5K，对于平常就有慢跑习惯的他，轻轻松松的就能完赛；就首次参与ELLE路跑活动的他非常兴奋，前一晚还一度失眠，但能早起运动对现在的他非常重要，不但能维持健康，也能开始训练之后宣传期所需的体力。</p>
<figure id="attachment_11201160" style="width: 451px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2019/04/1904201236451487.jpg"><img class=" wp-image-11201160" title="ELLE Run with Style" src="http://i.epochtimes.com/assets/uploads/2019/04/1904201236451487.jpg" alt="ELLE Run with Style" width="451" b="677" /></a><figcaption class="wp-caption-text">蔡F佑于4月20日出席“ELLE Run with Style风格路跑”活动。（ELLE提供）</figcaption></figure>
<p>即将在暑假发片的蔡F佑，表示近几个月忙于创作以及录音，疏忽了运动；但为了保持录音的最佳状态，还是会尽量维持跑步习惯。</p>
<p>他提到这次专辑睽违了将近四年，其实经过了创作空窗期、自信全失、再度振作等过程，最后才终于交出他认可的作品；也可能因为压力太大，让他半夜还会作恶梦。而接下来，他将全心投入新专辑的录音，暑假前就能带着新专辑跟大家见面。</p>
