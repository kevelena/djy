<tr><td VALIGN=TOP width="626"><h2>GOT7迷你九辑摘17区iTunes专辑榜冠军</h2>
<img src="http://i.epochtimes.com/assets/uploads/2019/05/190522042958100707-600x400.jpg" />
<h6>韩国人气男团GOT7迷你九辑《SPINNING TOP : BETWEEN SECURITY &#038; INSECURITY》宣传照。（环球唱片提供）
</h6>
<hr>
<p>【大纪元2019年05月22日讯】（大纪元记者曾薇心综合报导）韩国人气男团GOT7于5月20日发行的第九张迷你专辑《SPINNING TOP : BETWEEN SECURITY &amp; INSECURITY》，在21日的Hanteo Chart单日专辑榜拿下冠军。该作还于香港、泰国、巴西等17个地区的iTunes专辑榜摘冠。</p>
<p>GOT7的《SPINNING TOP : BETWEEN SECURITY &amp; INSECURITY》在收录取的歌词中融入了自传式的故事，此专辑也是从队长JB的一句“旋转的陀螺感觉就像我的样子”开始的。专辑中涵带了这样的概念：“人生中既有成功抓住重心、自信旋转的‘光明’瞬间，也有让安定的状况动摇的‘黑暗’瞬间共存。”</p>
<figure id="attachment_11272481" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2019/05/190522042951100707.jpg"><img class="size-large wp-image-11272481" title="GOT7" src="http://i.epochtimes.com/assets/uploads/2019/05/190522042951100707-600x600.jpg" alt="GOT7" width="600" b="600" /></a><figcaption class="wp-caption-text">GOT7迷你九辑《SPINNING TOP : BETWEEN SECURITY &amp; INSECURITY》封面照。（环球唱片提供）</figcaption></figure>
<p>另外，GOT7于20日韩国时间晚间6点公开的主打歌《ECLIPSE》官方MV，也在今天（22日）韩国时间上午1点左右突破1千万次观看。</p>
<p>主打歌《ECLIPSE》由JYP制作人朴轸永作词、队长JB（Defsoul）参与作词作曲。本次GOT7的专辑以<a href="https://github.com/szzd1/djy/blob/master/gb/tag/%E6%9C%89%E8%B0%A6.md">有谦</a>参与作词、以旋转陀螺为原型的《Degree》开始，到<a href="https://github.com/szzd1/djy/blob/master/gb/tag/%E7%8F%8D%E8%8D%A3.md">珍荣</a>参与作词作曲的抒情歌曲《The End》，前半的三首歌曲表达了从光亮到黑暗袭来的瞬间。后半张专辑，<a href="https://github.com/szzd1/djy/blob/master/gb/tag/%E8%8D%A3%E5%AE%B0.md">荣宰</a>（Ars）的《TIME OUT》、BamBam参与的《Believe》，以及JB的《PAGE》则展现了从黑暗中寻找光明的时刻。</p>
<figure id="attachment_11272480" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2019/05/190522042954100707.jpg"><img class="size-large wp-image-11272480" title="GOT7" src="http://i.epochtimes.com/assets/uploads/2019/05/190522042954100707-600x400.jpg" alt="GOT7" width="600" b="400" /></a><figcaption class="wp-caption-text">GOT7迷你九辑《SPINNING TOP : BETWEEN SECURITY &amp; INSECURITY》宣传照。（环球唱片提供）</figcaption></figure>
<p>2018年GOT7在全球17个城市举办了共计21次公演的世界巡演，今年他们的世界巡回演唱会“GOT7 2019 WORLD TOUR KEEP SPINNING”将于6月15日及16日起自韩国首尔奥林匹克体操竞技场（KSPO DOME）展开，他们也已公布亚洲、北南美以及欧洲等首轮世界巡回演唱会的演出地。</p>
<p>而GOT7的迷你九辑《SPINNING TOP : BETWEEN SECURITY &amp; INSECURITY》专辑也已在台湾数位发行。</p>
